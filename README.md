# Event Collector 🚀

這是一個基於 React 的前端應用程式，旨在提供一個整合的平台來管理和收集活動資訊。它利用 Tesseract.js 進行光學字元辨識 (OCR)，能夠從圖片中掃描並提取活動票券的詳細資料，並結合 Google Generative AI 的能力，進一步智慧化處理或分類這些事件資訊。使用者可以透過日曆視圖瀏覽活動、手動輸入資料，或利用掃描功能快速數位化票券。
